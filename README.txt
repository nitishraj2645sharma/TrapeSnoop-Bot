
# Telegram OSINT Bot

## Features
- Forwards all user messages to admin group.
- Shows sender's username and ID.
- Admins can reply to users from group.
- Ban/unban users using /ban or /unban.
- Get total user count with /stats.

## Commands
- /ban <user_id>
- /unban <user_id>
- /stats

## To Run
```
pip install -r requirements.txt
python main.py
```
